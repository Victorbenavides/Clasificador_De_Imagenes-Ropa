# Clasificador_De_Imagenes-Ropa
Este código es la base para crear un sistema de Inteligencia Artificial que aprende a reconocer ropa automáticamente. En lugar de darle órdenes a la computadora sobre cómo es una camisa, le enseñas miles de fotos para que ella misma aprenda a distinguirlas.
